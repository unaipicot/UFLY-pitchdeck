Core Technology

UFLY AI

The AI Pilot: Deep Reinforcement Learning

Our AI doesn't follow pre-programmed waypoints like traditional autopilots. It learns to fly from scratch using Proximal Policy Optimization (PPO), mastering the full 4D control space.

🧠 Custom Physics Engine

Standard simulators lack fidelity. We built a custom aerodynamic engine with non-linear CL-α stall modeling for the NACA 4412 airfoil — so the AI learns realistic stall recovery.

🎮 4D Control Space

Ailerons, Elevator, Rudder, Throttle — all continuously controlled by the neural network. Action low-pass filters (α=0.18) prevent actuator hard-overs.

📡 20-Dim Observation Space

EASA-compliant virtual sensor suite: IMU (100Hz), Pitot (20Hz), Barometric altimeter, GPS (5Hz). Same sensor inputs as certified avionics.

3-Phase Curriculum Learning

Phase 1 Stabilization

200K steps — Wings-level flight, attitude control, basic survival.

Phase 2 Navigation

22-episode skill blocks, progressive turns from 20° to 60° banked coordinated turns.

Phase 3 Mission Execution

Multi-waypoint routes, wind disturbances, domain randomization for real-world transfer.
